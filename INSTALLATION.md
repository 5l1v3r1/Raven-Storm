## Running:
`git clone https://github.com/Taguar258/Raven-Storm`

`cd Raven-Storm`

pip2 install re redis

### Stable but no ddos:
`python2 dos-stable.pyc`

### DDos version (newest):
`python2 dos.pyo`

Type `help` for help.

## Install updates:
`chmod 777 update.sh`

`./update.sh`

#### or:


`python2 dos.pyc`

`update`
